Lier un champ FK_

Pour l'exemple, j'ai ajouté une table t_sexe

Ainsi qu'un champ fk_sexe à la fin de ma table t_personne

Pour définir le FK, cliquez sur index du champs fk

Un pop-up apparait, cliquez sur OK pour confirmer.

Une petite clé grise est apparue à côté du champ

Passez en vue relationnelle :

Dans la partie Contraintes de clé étrangère,

  1. Le champ fk (ici fk_sexe)
  2. La table à lier (ici t_sexe)
  3. L'id (devrait se remplir tout seul donc pas besoin de toucher)
  4. Cliquez sur Sauvegarder et votre relation est terminé
