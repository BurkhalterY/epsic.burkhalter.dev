Les types de balises

Il existe deux types de balises : block et inline. Voici un petit tableau récapitulatif :

Block Inline
Taille Occupe toute la largeur N'occupe que la largeur nécessaire
Position L'un en dessous de l'autre Les deux côte-à-côte
Exemple de balise

Mais en pratique, à quoi servent ces balises ? Voici un tableau avec les principales balises à connaître :

Balise Inline/block Description
Block C'est la balise block "de base", elle permet de séparer les éléments pour, par exemple, leurs appliquer une classe CSS.
Inline Équivalent de en inline.
Block Un paragraphe. Pour insérer un retour à la ligne, écrire .
, ... Block Les titres et sous-titres du plus important (h1) au moins important (h6).
Inline Un lien hypertexte.
Block Une image.
Block Une liste.
Block Un tableau.
Inline Des champs d'entrée.
