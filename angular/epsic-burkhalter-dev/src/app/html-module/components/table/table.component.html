Les tableaux

Un tableau se constitue au minimum de 3 éléments : , et .

  • : c'est à l'intérieur qu'on va mettre nos lignes.
  • : pour Table Row, c'est une ligne du tableau.
  • : pour Table Data, c'est une cellule du tableau.

(Table Header) peut parfois être utilisé à la place de lorsqu'on veut indiquer qu'une cellule fait partie de l'entête.
