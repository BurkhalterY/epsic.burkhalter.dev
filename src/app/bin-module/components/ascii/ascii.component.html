Code ASCII

L'acronyme ASCII signifie American Standard Code for Information Interchange (Code américain normalisé pour l'échange d'information).

  • L'ASCII se code sur 7 bits, soit 128 caractères.
  • L'ASCII étendu se code sur 8 bits, soit 256 caractères.
