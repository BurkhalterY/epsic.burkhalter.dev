Les listes

Il existe deux types de liste: les listes non ordonnées (unordered) et les listes ordonnées (ordered).

Les listes qu'on utilise le plus fréquemment sont les listes non ordonnées. Elle se code comme ceci avec une balise qui contiendra tous les éléments de la liste entre balise :
  • Pierre
  • Paul
  • Jacques

Pour utiliser une liste ordonnée, simplement utiliser à la place de . Les numéros seront générés automatiquement :
  1. Pierre
  2. Paul
  3. Jacques

Listes imbriquées

Il est également possible de mettre une liste dans une autre afin d'avoir une sous-liste :
  • Titre
  • Menu 1
    • Sous-menu 1
    • Sous-menu 2
  • Menu 2
